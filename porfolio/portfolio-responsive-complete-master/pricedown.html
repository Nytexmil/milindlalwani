Pricedown

Know when the price is down

This was one of my very first projects. It was intended to track the price of products until the product was available in an affordable range. I used the soup library in Python to webscrap certain ecommerce sites to track the prices of their products. This provides a different service from what something like Honey does today in coupon searching. A few limitations were that I couldn't apply it to every ecommerce site, as each one had different tags and ids for which their products were under, but nonetheless, this was a great application of Python. I'm currently revising the code, but reach out if you would like the source code as this is on my old system and not github.
